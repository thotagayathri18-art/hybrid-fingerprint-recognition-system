\# Hybrid Fingerprint Recognition System



A desktop fingerprint recognition system built with Python, OpenCV, Tkinter, and PyTorch.



\## Features



\- Login authentication

\- Fingerprint enrollment

\- ORB-based fingerprint matching

\- CNN deep learning prediction

\- Hybrid recognition system

\- Criminal database integration

\- Webcam capture

\- Live recognition

\- Match history logging

\- PDF report export

\- Dashboard statistics



\## Technologies Used



\- Python

\- OpenCV

\- Tkinter

\- PyTorch

\- ReportLab

\- Pillow

\- CSV database
